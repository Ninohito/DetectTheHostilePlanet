# DetectTheHostilePlanet
Pyxelではじめて作ったゲームです。
近くに敵対する惑星がありますが、基本的に不可視です。
文明を強化し、Eキーで敵対惑星の動きを推定して攻撃を回避、または撃退してください。
Generation Points(GP)を消費して惑星を追加するか既存の惑星を強化することができます。
体力(HP)が0以下になるとGPが生成されなくなるので注意してください。（ゲームの続行は可能です。）

基本的な操作方法は画面上部に表示されます。
カーソルを惑星に合わせると、惑星ごとの機能と対応する操作キーが表示されます。

辞書：
Hostile: 敵対的
Threaten: 脅かす
Strengthen: 強化する
Civilization: 文明
Expand: 拡大する
Power: 力
Defeat: 倒す

Earth: 地球
Venus: 金星
Mars: 火星
Mercury: 水星
Ceres: ケレス
