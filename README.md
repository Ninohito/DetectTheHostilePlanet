# DetectTheHostilePlanet
Pyxelではじめて作ったゲームです。</p>
</p>

## [開始するにはここをクリック](https://ninohito.github.io/DetectTheHostilePlanet/)

近くに敵対する惑星がありますが基本的に不可視です。</p>
Eキーで敵対惑星の動きを推定して攻撃を回避、または撃退してください。</p>
Generation Points(GP)を消費して惑星を追加するか既存の惑星を強化することができます。</p>
体力(HP)が0以下になるとGPが生成されなくなるので注意してください。（ゲームの続行は可能です。）</p>
カーソルを惑星に合わせると、画面上部に惑星ごとの機能と対応する操作キーが表示されます。
